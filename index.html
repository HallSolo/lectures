Типы данных

Числовые типы данных

Целые числа.

Dim b As Byte
Предел значений между 0 и 255

Dim k As Integer
Предел значений между -2,147,483,648 и +2,147,483,647

Dim l As Long
Предел значений между -9,223,372,036,854,775,808 и 9,223,372,036,854,775,807

Значения по-умолчанию 0

Числовые типы данных

Действительные числа.

Dim s As Single
Предел значений между -3.4028235E+38 и -1.401298E-45
или 1.401298E-45 и 3.4028235E+38

Dim d As Double
Предел значений между -1.79769313486231570E+308 и -4.94065645841246544E-324
или 4.94065645841246544E-324 и 1.79769313486231570E+308

Dim dc As Decimal
Предел значений между +/-79,228,162,514,264,337,593,543,950,335

Значения по-умолчанию 0

Строковые типы данных




Dim j As String = "Joe said ""Hello"" to me."
Сохраняет строку длинной от 0 до $2^{31}$ символов Unicode

Значение по-умолчанию Nothing



Dim charVar As Char
Сохраняет 1 символ Unicode

Значение по-умолчанию 0

Специальные типы данных




Dim runningVB As Boolean
Сохраняет логические значения False и True

Значение по-умолчанию False



Dim myObject As Object
Сохраняет экземпляр класса

Значение по-умолчанию Nothing



Dim MyVar As Variant
Сохраняет значения не определенного типа
