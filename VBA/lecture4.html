Основы объектно-ориентированного программирования

Класс

— блок функциональности, которые можно использовать в программах.




Объект

— экземпляр конкретного класса.

Создание объектов



Позднее связывание 

		  Dim oApp As Object
		  Set oApp = CreateObject("Word.Application")
		  MsgBox oApp.UserName


Ранее связывание 

		  Dim oApp As Object
		  MsgBox oApp.UserName

Удаление объектов





		Set oDoc = Nothing

Методы объекта



— именованный набор действий, которые может выполнять данный объект.

Варианты вызова методов: 

		  oDoc.Activate
	      

		  oDoc.SaveAs "C:\doc12.doc"
	      

Dim nCent
nCent = oApp.CentimetersToPoints(10)

Свойства объекта



— это информация, которая хранится в этом объекте и к которой можно получить доступ. 


		oApp.ActivePrinter = "HP LaserJet 4"

События объекта



— это действие, распозноваемое объектом, для которого можно запрограммировать реакцию. 


		Public WithEvents App As Word.Application

Range

— набор ячеек.

Инициализация c 1 ячейкой: 

Dim oRange As Range
Set oRange = Worksheets("Лист1").Range("A1")


Инициализация с диапозоном: 

Dim oRange As Range
Set oRange = Worksheets("Лист1").Range("A1:D10")

Свойства объекта Range

  • Width()
    • — ширина ячеек.
  • Height()
    • — высота ячеек.
  • Font()
    • — шрифт в ячейках.
  • Formula
    • — формула вычисления значения ячеек.
  • Borders
    • — ссылка на границу набора ячеек.

Методы объекта Range

  • Cut()
    • — вырезать содержание ячеек.
  • Delete()
    • — удаление данных из ячеек диапозона.
  • Clear()
    • — очистить ячейки диапозона.
  • Merge()
    • — объединение всех ячеек в одну.
  • UnMerge()
    • — обратная операция.
  • Sort()
    • — сортировать содержание ячеек.

Пример использования




Sub main()

   Dim MyRange As Range
   Dim myBorder As Borders

   Set MyRange = Range("B2:C3")
   Set myBorder = MyRange.Borders

   myBorder.Color = vbRed

End Sub

Worksheets

— лист рабочей книги Excel.

Инициализация с добавлением: 

Dim oSheet As Excel.Worksheet
Set oSheet = Worksheets.Add()

Свойства объекта Worksheet

  • Next
    • — ссылка на следующий лист.
  • Previous
    • — ссылка на предыдущий лист.
  • Name
    • — наименование листа.
  • Type
    • — тип данного листа.
  • Visible
    • — скрытость от глаз пользователя.

Методы объекта Worksheet

  • Copy()
    • — копирование листа.
  • Delete()
    • — удаление листа.
  • Move()
    • — перемещение листа.
  • Calculate()
    • — перевычисление листа.
  • SaveAs()
    • — сохранение листа в файл.
  • Protect()
    • — защита листа от изменений.

Пример использования




Sub main()

    Dim oWorksheets As Worksheet
    Dim strPassword As String 

    Set oWorksheets = Worksheets("Лист1")

    strPassword = InputBox ("Введите пароль:") 
    oWorksheets.Protect password:=strPassword, scenarios:=True

End Sub

События объекта Worksheet



  • Activate
    • — активирование листа.
  • Change
    • — изменение на листе.
  • Calculate
    • — вычисление листе.

Пример использования





Private Sub Worksheet_Change(ByVal Target As Range)
        If Target.Column = 1 Then
        ThisRow = Target.Row
        If Target.Value > 100 Then
            Range("B" & ThisRow).Interior.ColorIndex = 3
        Else

        End If
    End If
