Операции сравнения

Обозначения операций сравнения


= E1 = E2 Равно.
< E1 < E2 Меньше, чем.
<= E1 <= E2 Меньше или равно.
> E1 > E2 Больше, чем.
>= E1 >= E2 Больше или равно.
<> E1 <> E2 Не равно.
Is E1 Is E2 Идентичность.
Like E1 Like E2 Подобие.

Символы совпадения с образцом


# Любая одиночная цифра от 0 до 9.
* Любое количество символов в любой комбинации.
? Любой одиночный символ.
[list] Совпадение с любым одиночным символом из списка.
[!list] Совпадение с любым одиночным символом, не имеющемся в списке.

Управляющие операторы

Условный оператор

If...Then...Else

    Dim count As Integer = 0
    Dim message As String
                            

    If count = 0 Then
        message = "There are no items."
                            

    ElseIf count = 1 Then
        message = "There is 1 item."
                            

    Else
        message = "There are " & count & " items."
                            

    End If

Блок-схема

Условный оператор

Select...Case

    Dim number As Integer = 8
                            

    Select Case number
        Case 1 To 5
            Debug.WriteLine("Between 1 and 5, inclusive")
Проверочное выражение должно принадлежать простому типу данных
Список выражений может состоять:
  • выражений
  • диапазона выражений 1 To 5
  • выражением сравнения Is > 5
 

    Case 6, 7, 8
            Debug.WriteLine("Between 6 and 8, inclusive")
    Case 9 To 10
            Debug.WriteLine("Equal to 9 or 10")
                            

    Case Else
            Debug.WriteLine("Not between 1 and 10, inclusive")
                            

    End Select

For...Next


    For index As Integer = 1 To 5
        Debug.Write(index.ToString & " ")
    Next
    Debug.WriteLine("")
    ' Output: 1 2 3 4 5

Цикл с определенным шагом


    For number As Double = 2 To 0 Step -0.25
        Debug.Write(number.ToString & " ")
        Next
        Debug.WriteLine("")
        ' Output: 2 1.75 1.5 1.25 1 0.75 0.5 0.25 0

Управление циклом


    For index As Integer = 1 To 100000
                                    

    ' If index is between 5 and 7, continue
    ' with the next iteration.
    If index >= 5 And index <= 8 Then
        Continue For
    End If
                                    

    ' Display the index.
    Debug.Write(index.ToString & " ")
                                    

    ' If index is 10, exit the loop.
    If index = 10 Then
        Exit For
    End If
                                    

    Next
                                    

    Debug.WriteLine("")
    ' Output: 1 2 3 4 9 10
